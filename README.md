# 🍕 Pizza Cafe Website

A responsive pizza restaurant website built using **HTML, CSS, and Vanilla JavaScript**. It showcases a modern UI with sections for menu, specials, reviews, and ordering.

🔗 **Live Demo:** https://pizzacafe.vercel.app/

## 🚀 Features

* Responsive design
* Pizza menu display
* Specials and offers section
* Customer reviews
* Simple interactive UI with JavaScript

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## 📂 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
